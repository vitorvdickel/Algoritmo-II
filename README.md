# 📚 Algoritmo II - IFSUL

Repositório dedicado aos estudos, exercícios e trabalhos práticos da disciplina de **Algoritmos II** do curso de Ciência da Computação no Instituto Federal Sul-rio-grandense (IFSUL) - Campus Passo Fundo.

## 🎯 Objetivo do Repositório
O objetivo principal aqui é documentar o meu avanço no aprendizado de Estruturas de Dados e Programação Orientada a Objetos, mantendo um histórico das resoluções de problemas propostos em aula.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** C++
* **Ambiente de Desenvolvimento:** Visual Studio Code (VS Code)

## 📂 Estrutura do Projeto
Os arquivos estão organizados da seguinte forma:
* `/aulas`: Códigos e anotações desenvolvidos durante as explicações em sala de aula.
* `/listas`: Resoluções das listas de exercícios propostas pelo professor.

## 🚀 Como executar os códigos
Para compilar e rodar qualquer um dos exercícios localmente, você precisará de um compilador C++ (como o GCC/MinGW) instalado. 

Pelo terminal, navegue até a pasta do arquivo e execute:
```bash
g++ nome_do_arquivo.cpp -o executavel
./executavel  # No Windows, use apenas executavel.exe
