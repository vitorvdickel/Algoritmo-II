vetor com função
para manipular um vetor utilizando função identicando como um vetor
através do uso dos colchetes, não necessita informar a dimensão, pois 
ao adicionar a função estaremos referenciando ao endereço de memória do vetor
original.
EX:
void leitura(int x[])

para acionar a função devemos informar o endereço de memoria onde
inicia o vetor
EX:
int a[10];
leitura (a);
leitura (&a[0]);

main(){

    leitura(a);
}

Matriz com função
Para manipular um array de duas dimenções (matriz) através de uma função
deve-se declarar o parâmetro da função informando o número de colunas
que a matriz vai conter
EX:
void leitura(int mat[][5]);
a chamada deve ser passando somente o nome da variável original
leitura(a);

main(){
    int a[5][5];
    leitura (a);
}
void leitura(int mat[][5]){
    int i, x;
    srand(time(NULL));
    for (i = 0; i < 5; i++){
        for (x = 0; x < 5; x++){
            mat[i][x] = rand()%100
        }
    }
}
